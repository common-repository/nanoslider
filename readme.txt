== Installation ==

To install the plugin manually:

1.Download NanoSlider app for Win or Mac from  
   http://shop.android-apps-tablet.com/nano-slider.zip
2. Extract the contents of the archive (zip file)
3. Upload the nano-slider folder to your '/wp-content/plugins' folder
4. Activate the plugin through the Plugins section in your WordPress admin

Usage in Content: [nano-slider]
Usage in template file: echo do_shortcode('[nano-slider]');
